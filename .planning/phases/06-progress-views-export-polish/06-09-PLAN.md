**Encoder configuration locked in this plan (referenced by 06-09b):**
```
extension JSONEncoder {
    public static func exportConfigured() -> JSONEncoder {
        let enc = JSONEncoder()
        enc.outputFormatting = [.prettyPrinted, .sortedKeys, .withoutEscapingSlashes]
        enc.dateEncodingStrategy = .iso8601
        return enc
    }
}
```
Add this extension inside ExportDTOs.swift so the snapshot test, the eventual JSONExporter (06-09b), and the BackupArchiver (06-10) all share one configuration source-of-truth.
