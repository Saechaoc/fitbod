internal enum ZIPWriter {
    /// Compose a PKWARE-format ZIP archive from entries. STORE method baseline;
    /// DEFLATE via Compression framework when payload >= 64 bytes (planner choice).
    /// Returns the assembled archive bytes.
    static func write(entries: [Entry]) throws -> Data

    struct Entry {
        let path: String      // forward-slash, no leading slash, e.g. "manifest.json" or "images/{uuid}.jpg"
        let data: Data        // uncompressed payload
        let lastModified: Date
    }
}

**ZIPReader shape (new file `fitbod/Backup/ZIPReader.swift`):**
```
internal enum ZIPReader {
    /// Parse a PKWARE-format ZIP archive. Validates CRC-32 per entry.
    /// Throws on malformed records, CRC mismatch, or size > maxTotalUncompressed.
    static func read(data: Data, maxTotalUncompressed: Int = 100 * 1024 * 1024) throws -> [Entry]

    struct Entry {
        let path: String
        let data: Data
    }

    enum ReadError: Error, Equatable {
        case missingEndOfCentralDirectory
        case malformedCentralDirectory
        case malformedLocalFileHeader
        case unsupportedCompressionMethod(UInt16)
        case crc32Mismatch(path: String)
        case totalSizeExceeded
    }
}
```

**ZIP format reference (PKWARE APPNOTE 6.3.10) — Required records:**
- **Local File Header** (signature `PK\x03\x04` = 0x04034b50): version=20, gpFlag=0, compressionMethod (0=STORE / 8=DEFLATE), lastModTime/Date (DOS format), crc32, compressedSize, uncompressedSize, fileNameLength, extraFieldLength, fileName, extraField (empty), then payload bytes.
- **Central Directory Header** (signature `PK\x01\x02` = 0x02014b50): versionMadeBy=20, versionNeeded=20, gpFlag=0, compressionMethod, lastModTime/Date, crc32, compressedSize, uncompressedSize, fileNameLength, extraFieldLength, commentLength=0, diskNumberStart=0, internalFileAttrs=0, externalFileAttrs=0, localHeaderOffset (offset of corresponding local file header from start of archive), fileName.
- **End of Central Directory** (signature `PK\x05\x06` = 0x06054b50): diskNumber=0, centralDirStartDisk=0, totalEntriesOnThisDisk=N, totalEntries=N, centralDirSize, centralDirOffset, commentLength=0.

All multi-byte integers little-endian. CRC-32 uses PKWARE polynomial (0xEDB88320 reflected) — implement a 256-entry table once at writer init; zlib's `crc32` from `Compression` framework's underlying zlib also works if exposed.

**DEFLATE via Compression framework** (when archiver chooses to compress an entry — planner: use STORE for `manifest.json` and `store.json` if simpler; OK to use STORE for all entries in v1):
```
let compressed = (try? (payload as NSData).compressed(using: .zlib).deflated()) ?? payload   // sketch only
```
The simpler v1 path: **STORE-only (compressionMethod=0)**. CONTEXT D-30 does not require compression; ZIP-format conformance is the load-bearing requirement. Pick STORE for all three entry kinds and skip the Compression-framework dance. The result is still a Finder-readable ZIP.

**Recommendation locked:** Use compressionMethod=0 (STORE) for all entries in v1. If profiling later shows backup size is a problem, swap to DEFLATE for `store.json` only — single-call-site change inside `ZIPWriter.write`.

**Security gates (RESEARCH §Security Domain — Known Threat Patterns):**
1. **Path traversal:** every archived entry's path must start with `images/` or be exactly `manifest.json` / `store.json` AND must not contain `..` AND must not start with `/` — reject `.pathTraversalDetected`. Enforced in ZIPReader during parse (cheap) AND again in BackupRestorer before any disk write (defense in depth).
2. **Decompression bomb:** total uncompressed bytes across all entries capped at 100 MB — reject `.decompressionBombExceeded`. Enforced inside ZIPReader.read via `maxTotalUncompressed` parameter (default 100 * 1024 * 1024).
3. **Checksum mismatch:** SHA-256 of decoded `store.json` must equal `manifest.checksum` — reject `.checksumMismatch`.
4. **Schema-skew:** `manifest.schemaVersion == ExportDocument.canonicalSchemaVersion` — exact string equality — reject `.schemaVersionMismatch`.
5. **Active session blocker:** preflight queries `Session` where `completedAt == nil`; if any exist, reject `.activeSessionInProgress`.
