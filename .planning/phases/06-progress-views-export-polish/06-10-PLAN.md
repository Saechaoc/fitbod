**BackupRestorer (PATTERNS §"Backup/BackupRestorer.swift" + RESEARCH §"Pattern 6" + §Security Domain):**
```
@MainActor
public final class BackupRestorer {
    public enum RestoreError: Error, Equatable {
        case fileUnreadable
        case checksumMismatch
        case schemaVersionMismatch(found: String, expected: String)
        case activeSessionInProgress
        case decompressionBombExceeded
        case pathTraversalDetected(entry: String)
        case decodeFailed(underlying: String)
    }

    public init() {}

    /// Step 1: read the archive at url; verify manifest checksum + schemaVersion + no active session; return BackupManifest on success.
    public func preflight(url: URL, context: ModelContext) async throws -> BackupManifest

    /// Step 2: after the destructive-confirmation alert, perform restore:
    ///   - Write current container's full export to documentDirectory/store-pre-restore-{timestamp}.json (side-file rollback per D-32 step 4).
    ///   - Delete the on-disk SQLite store files of the current container.
    ///   - Instantiate a fresh ModelContainer.
    ///   - Decode store.json into ExportDocument; insert all entities; commit.
    ///   - Copy images/{uuid}.jpg blobs back into Exercise.imageData for matching exerciseIDs.
    public func performRestore(url: URL, manifest: BackupManifest, currentContainer: ModelContainer) async throws

    /// Step 3: caller (Settings view) presents the "Restore complete" alert; OK button calls this.
    public func restart() -> Never { exit(0) }
}
```

**Security gates (RESEARCH §Security Domain — Known Threat Patterns):**
1. **Path traversal:** every archived entry's path must start with `images/` or be exactly `manifest.json` / `store.json` AND must not contain `..` — reject `.pathTraversalDetected`
2. **Decompression bomb:** store.json bytes capped at 100 MB — reject `.decompressionBombExceeded`
3. **Checksum mismatch:** SHA-256 of decoded store.json must equal `manifest.checksum` — reject `.checksumMismatch`
4. **Schema-skew:** `manifest.schemaVersion == ExportDocument.canonicalSchemaVersion` — exact string equality — reject `.schemaVersionMismatch`
5. **Active session blocker:** preflight queries `Session.active(in: context)` (or equivalent — see existing SessionFactory.active helper); reject `.activeSessionInProgress`
