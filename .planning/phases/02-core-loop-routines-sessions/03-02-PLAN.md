# Plan 03-02 — Routine Builder, Prescription Editor, and Inline Exercise Picker

> **Wave 3 / Sequence 2.** Ships `RoutineBuilderView` — the single-screen builder per ROUTINE-01 (no modal exercise picker). Refactors `ExerciseLibraryView` to add the `init(onSelect: (Exercise) -> Void)` overload from RESEARCH § Pattern 5 so the picker can be embedded inline at the bottom of the builder list. Adds `PrescriptionEditorRow` (inline expanded per-exercise editor: intent / sets / reps range / RPE range / progression kind / rest seconds / tempo toggle / per-set overrides disclosure) and `PerSetOverrideRow`. Adds `PrescriptionDefaults` for the mechanic/equipment heuristic (ROUTINE-09: compound → 180s rest, isolation → 90s; barbell compound → strength intent, otherwise hypertrophy). Closes ROUTINE-01, ROUTINE-02, ROUTINE-03 (data path), ROUTINE-05, ROUTINE-09, SESS-07 (toggle), SESS-08 (toggle).

## Goal

Tap "New Routine" in the Routines tab → `RoutineBuilderView` pushed onto the Routines tab's `NavigationStack`. The user names the routine, picks a folder (or leaves Unfiled), and adds exercises via an inline search bar at the bottom of the list (reusing Phase 1's `ExerciseLibraryView` in picker mode). Each row is a `RoutineExerciseCard` showing the exercise name + intent chip + collapsed prescription summary; tap to expand the inline `PrescriptionEditorRow`. Drag handle (`line.3.horizontal`) on the right always-visible (`.environment(\.editMode, .constant(.active))`); `.onMove` rewrites `orderIndex` on every reorder (RESEARCH §6 Pitfall 10). Per-set overrides are sub-rows under the prescription editor's "Per-set overrides" disclosure. Per-exercise toggles for `tracksTempo` and `tracksPartialReps` opt the user into the tempo + partial-reps rows in the logger (SESS-07 / SESS-08 — the visual toggles live here; the row rendering lives in plan `04-02`).

When `targetSets` decreases, the draft prunes per-set overrides where `setIndex >= newTargetSets` (Pitfall 8).

## Requirements Covered

- **ROUTINE-01** (`single-screen builder with inline search-and-add, drag-reorder, no modal exercise picker`) — the canonical UI surface.
- **ROUTINE-02** (`each exercise carries first-class intent / target rep range / target RPE range`) — exposed in `PrescriptionEditorRow`.
- **ROUTINE-03** (per-set prescription overrides) — `PerSetOverrideRow` + "Per-set overrides" disclosure + add/remove logic + Pitfall 8 prune-on-targetSets-shrink.
- **ROUTINE-05** (`progression model per exercise from four options`) — the Progression picker (RPE Autoregulation / Double Progression / Block Periodized / Hybrid).
- **ROUTINE-09** (`default rest timer heuristic by mechanic`) — `PrescriptionDefaults.applyDefaults(to:)` runs when a new exercise is appended.
- **SESS-07** (optional tempo per set) — the `Track tempo` toggle + 4-field ecc/bot/con/top entry row in the prescription editor; the per-set logger rendering lives in plan `04-02`.
- **SESS-08** (partial reps + cluster sub-reps) — the `tracksPartialReps: Bool` toggle in the prescription editor (the partials row in the logger lives in plan `04-02`).

ROUTINE-04 (supersets) is intentionally deferred to plan `03-03` — it requires a separate `SupersetAssignmentSheet` + visual rail logic that this plan keeps out of scope to avoid bloat.

```
import Foundation
import SwiftData

@Observable
@MainActor
public final class RoutineDraft {
    public var name: String = ""
    public var notes: String? = nil
    public var folderID: UUID? = nil
    public var exercises: [RoutineExerciseDraft] = []

    public init() {}

    public init(routine: Routine) {
        self.name = routine.name
        self.notes = routine.notes
        self.folderID = routine.folderID
        self.exercises = (routine.exercises ?? [])
            .sorted { $0.orderIndex < $1.orderIndex }
            .map { RoutineExerciseDraft(re: $0) }
    }

    public var isValid: Bool {
        let trimmed = name.trimmingCharacters(in: .whitespacesAndNewlines)
        return !trimmed.isEmpty && !exercises.isEmpty
    }

    /// Append a new exercise to the draft. Applies the
    /// `PrescriptionDefaults` heuristic per ROUTINE-09.
    public func append(exercise: Exercise) {
        let draft = RoutineExerciseDraft()
        draft.exercise = exercise
        PrescriptionDefaults.apply(to: draft, from: exercise)
        draft.orderIndex = exercises.count
        exercises.append(draft)
    }

    /// Persist the draft to a Routine entity (insert if new, update if
    /// editing). Called from RoutineBuilderView's "Save" toolbar action.
    public func save(into routine: Routine, context: ModelContext) {
        routine.name = name.trimmingCharacters(in: .whitespacesAndNewlines)
        routine.notes = notes
        routine.folderID = folderID
        routine.updatedAt = .now

        // Remove existing RoutineExercise rows not in the draft.
        let existingExercises = routine.exercises ?? []
        let draftIDs = Set(exercises.compactMap { $0.id })
        for old in existingExercises where !draftIDs.contains(old.id) {
            context.delete(old)
        }

        for (i, draft) in exercises.enumerated() {
            let re: RoutineExercise
            if let id = draft.id, let match = existingExercises.first(where: { $0.id == id }) {
                re = match
            } else {
                re = RoutineExercise()
                re.routine = routine
                context.insert(re)
                draft.id = re.id
            }
            re.exercise = draft.exercise
            re.orderIndex = i
            re.intentRaw = draft.intent.rawValue
            re.targetSets = draft.targetSets
            re.targetRepsLow = draft.targetRepsLow
            re.targetRepsHigh = draft.targetRepsHigh
            re.targetRPE = draft.targetRPE
            re.targetRIR = nil
            re.prescribedRestSeconds = draft.prescribedRestSeconds
            re.progressionKindRaw = draft.progressionKind.rawValue
            re.tracksTempo = draft.tracksTempo
            re.tracksPartialReps = draft.tracksPartialReps
            re.tempo = draft.tempo
            re.supersetGroupID = draft.supersetGroupID
            // Persist per-set overrides (Pitfall 8 — prune-on-shrink
            // already happened in the draft layer; here we just write).
            let existingOverrides = re.setOverrides ?? []
            let draftSetIndices = Set(draft.setOverrides.map { $0.setIndex })
            for old in existingOverrides where !draftSetIndices.contains(old.setIndex) {
                context.delete(old)
            }
            for od in draft.setOverrides {
                let ov: RoutineExerciseSetOverride
                if let id = od.id, let match = existingOverrides.first(where: { $0.id == id }) {
                    ov = match
                } else {
                    ov = RoutineExerciseSetOverride()
                    ov.routineExercise = re
                    context.insert(ov)
                    od.id = ov.id
                }
                ov.setIndex = od.setIndex
                ov.targetRepsLow = od.targetRepsLow
                ov.targetRepsHigh = od.targetRepsHigh
                ov.targetRPE = od.targetRPE
            }
        }
    }
}

    public var exercise: Exercise? = nil

    public var intent: Intent = .hypertrophy
    public var targetSets: Int = 3 {
        didSet {
            // RESEARCH §6 Pitfall 8 — prune overrides on targetSets shrink.
            if targetSets < oldValue {
                setOverrides = setOverrides.filter { $0.setIndex < targetSets }
            }
        }
    }
    public var targetRepsLow: Int = 8
    public var targetRepsHigh: Int = 12
    public var targetRPE: Double? = 8.0
    public var prescribedRestSeconds: Int = 120
    public var progressionKind: ProgressionKind = .double
    public var tempo: String? = nil
    public var tracksTempo: Bool = false
    public var tracksPartialReps: Bool = false
    public var supersetGroupID: UUID? = nil
    public var setOverrides: [PerSetOverrideDraft] = []

    public init() {}

    convenience init(re: RoutineExercise) {
        self.init()
        self.id = re.id
        self.exercise = re.exercise
        self.orderIndex = re.orderIndex
        self.intent = re.intent
        self.targetSets = re.targetSets
        self.targetRepsLow = re.targetRepsLow
        self.targetRepsHigh = re.targetRepsHigh
        self.targetRPE = re.targetRPE
```

### Create — `fitbod/Routines/PrescriptionDefaults.swift`

Per CONTEXT.md Area 1 + ROUTINE-09.

```
import Foundation

public enum PrescriptionDefaults {
    /// Apply the mechanic/equipment heuristic to a freshly-added
    /// RoutineExerciseDraft per ROUTINE-09 + CONTEXT.md Area 1.
    ///
    /// - Compound → rest 180s
    /// - Isolation → rest 90s
    /// - Barbell + compound → strength intent
    /// - Otherwise → hypertrophy intent
    @MainActor
    public static func apply(to draft: RoutineExerciseDraft, from exercise: Exercise) {
        let isCompound = exercise.mechanic == .compound
        let isBarbell = exercise.equipment == .barbell

        draft.prescribedRestSeconds = isCompound ? 180 : 90

        if isCompound && isBarbell {
            draft.intent = .strength
            draft.targetRepsLow = 4
            draft.targetRepsHigh = 6
            draft.targetRPE = 8.0
            draft.progressionKind = .double
        } else {
            draft.intent = .hypertrophy
            draft.targetRepsLow = 8
            draft.targetRepsHigh = 12
            draft.targetRPE = 8.0
            draft.progressionKind = .double
        }
    }
}
```

```
import SwiftUI
import SwiftData

public struct RoutineBuilderView: View {
    @Environment(\.modelContext) private var ctx
    @Environment(\.dismiss) private var dismiss
    @Bindable public var draft: RoutineDraft

    /// nil = create mode; non-nil = edit mode (existing Routine).
    public let editing: Routine?

    @State private var expandedExerciseIDs: Set<UUID> = []
    @State private var presentingDiscardConfirm = false
    @State private var initialSnapshot: String = ""  // hash of draft at appear, for dirty-check

    public init(draft: RoutineDraft, editing: Routine? = nil) {
        self.draft = draft
        self.editing = editing
    }

    public var body: some View {
        Form {
            Section {
                TextField("Routine name", text: $draft.name)                  // UI-SPEC verbatim
            }
            Section("Exercises") {
                ForEach(Array(draft.exercises.enumerated()), id: \.offset) { index, exDraft in
                    RoutineExerciseCard(
                        draft: exDraft,
                        isExpanded: Binding(
                            get: { expandedExerciseIDs.contains(exDraft.id ?? UUID()) },
                            set: { isOpen in
                                let key = exDraft.id ?? UUID()
                                if isOpen { expandedExerciseIDs.insert(key) }
                                else { expandedExerciseIDs.remove(key) }
                            }
                        )
                    )
                }
                .onMove { source, destination in
                    draft.exercises.move(fromOffsets: source, toOffset: destination)
                    // RESEARCH §6 Pitfall 10 — rewrite orderIndex
                    for (i, ex) in draft.exercises.enumerated() {
                        ex.orderIndex = i
                    }
                }
                InlineExerciseSearchRow { exercise in
                    draft.append(exercise: exercise)
                }
            }
            if draft.exercises.isEmpty {
                Section {
                    Text("Add an exercise to begin.")                          // UI-SPEC verbatim empty body
                        .foregroundStyle(.secondary)
                }
            }
            Section {
                TextField("Notes (optional)", text: Binding(
                    get: { draft.notes ?? "" },
                    set: { draft.notes = $0.isEmpty ? nil : $0 }
                ), axis: .vertical)                                            // UI-SPEC verbatim placeholder
            }
        }
        .environment(\.editMode, .constant(.active))                           // always-on drag handles
        .navigationTitle(editing == nil ? "New Routine" : draft.name)          // UI-SPEC verbatim
        .toolbar {
            ToolbarItem(placement: .topBarLeading) {
                Button("Cancel") {
                    if hasUnsavedChanges {
                        presentingDiscardConfirm = true
                    } else {
                        dismiss()
                    }
                }                                                              // UI-SPEC verbatim
            }
            ToolbarItem(placement: .topBarTrailing) {
                Button("Save") {                                                // UI-SPEC verbatim
                    save()
                }
                .disabled(!draft.isValid)
                .foregroundStyle(Color.accentColor)                            // UI-SPEC accent surface #12
            }
        }
        .confirmationDialog(
            "Discard Changes?",                                                // UI-SPEC verbatim
            isPresented: $presentingDiscardConfirm,
            titleVisibility: .visible
        ) {
            Button("Discard", role: .destructive) {                            // UI-SPEC verbatim
                dismiss()
            }
            Button("Keep Editing", role: .cancel) {                            // UI-SPEC verbatim
                presentingDiscardConfirm = false
            }
        }
        .onAppear {
            initialSnapshot = snapshotHash()
        }
    }

    private var hasUnsavedChanges: Bool {
        snapshotHash() != initialSnapshot
    }

    private func snapshotHash() -> String {
        "\(draft.name)|\(draft.exercises.count)|\(draft.exercises.map { $0.targetSets }.reduce(0, +))"
    }

    private func save() {
        let routine = editing ?? Routine()
        if editing == nil {
            ctx.insert(routine)
        }
        draft.save(into: routine, context: ctx)
        try? ctx.save()
        dismiss()
    }
}
```

```
import SwiftUI

public struct RoutineExerciseCard: View {
    @Bindable public var draft: RoutineExerciseDraft
    @Binding public var isExpanded: Bool

    public init(draft: RoutineExerciseDraft, isExpanded: Binding<Bool>) {
        self.draft = draft
        self._isExpanded = isExpanded
    }

    public var body: some View {
        DisclosureGroup(isExpanded: $isExpanded) {
            PrescriptionEditorRow(draft: draft)
        } label: {
            HStack(spacing: 8) {
                VStack(alignment: .leading, spacing: 4) {
                    Text(draft.exercise?.name ?? "Exercise")
                        .font(.body)
                    HStack(spacing: 8) {
                        // Intent chip — UI-SPEC accent surface #15
                        Text(draft.intent.rawValue.capitalized)
                            .font(.caption)
                            .padding(.horizontal, 8)
                            .padding(.vertical, 4)
                            .background(Color.accentColor.opacity(0.15))
                            .foregroundStyle(Color.accentColor)
                            .clipShape(Capsule())
                        Text(prescriptionSummary)
                            .font(.caption)
                            .foregroundStyle(.secondary)
                    }
                }
            }
        }
        .tint(Color.accentColor)                                                // UI-SPEC accent surface #16 (chevron)
    }

    private var prescriptionSummary: String {
        let reps = draft.targetRepsLow == draft.targetRepsHigh
            ? "\(draft.targetRepsLow)"
            : "\(draft.targetRepsLow)–\(draft.targetRepsHigh)"
        return "\(draft.targetSets)×\(reps) · \(draft.prescribedRestSeconds)s"
    }
}
```

## Anti-Patterns to Avoid

- Do NOT introduce a `RoutineBuilderViewModel`. FOUND-06 / MV-VM-lite. `RoutineDraft` is the ephemeral `@Observable` mutation surface; the view binds to it via `@Bindable`.
- Do NOT mutate `RoutineExercise` rows directly from the editor. The builder edits the `RoutineExerciseDraft` mutable wrapper; mutations land on the `@Model` row only via `RoutineDraft.save(into:context:)` on Save tap.
- Do NOT call `try? ctx.save()` on every keystroke. Save only happens when the user taps the Save button; the draft is in-memory until then.
- Do NOT skip the `targetSets.didSet` prune (Pitfall 8). Without it, per-set overrides at indices >= newTargetSets linger as orphan rows.
- Do NOT skip the `.onMove` orderIndex rewrite (Pitfall 10). The visual reorder must persist; iterating the array and assigning `ex.orderIndex = i` is the canonical pattern.
- Do NOT remove the existing two `ExerciseLibraryView` init overloads. RESEARCH § Pattern 5 — the refactor is purely additive; Phase 1 callers must continue to work byte-for-byte.
- Do NOT render the inline picker as an inline `LazyVStack` inside the same `ScrollView` for v1. UI-SPEC describes inline rendering as the ideal; the sheet-presentation here is the pragmatic implementation that still reuses Phase 1's `ExerciseLibraryView`. Document the trade-off in SUMMARY.
- Do NOT auto-save on Cancel without prompting. UI-SPEC § Routine builder: the discard-changes confirmation dialog is mandatory when the form is dirty.
- Do NOT use a per-row sub-`@State` for the per-set override sub-rows. The overrides are bound to `PerSetOverrideDraft` via `@Bindable` directly; each sub-row binds to one draft.
- Do NOT compute the prescription-summary string ("\(targetSets)×\(reps) · \(rest)s") via `String(format:)` — use Swift string interpolation; the format string isn't load-bearing.
