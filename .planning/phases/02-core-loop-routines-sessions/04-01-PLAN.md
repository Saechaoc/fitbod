# Plan 04-01 — Session Logger View, Set Row, RPE Chips, Previous Column

> **Wave 4 / Sequence 1.** Ships the core session logger surface — `SessionLoggerView` (the workout-running screen), `SessionExerciseCard` (one card per snapshotted exercise), `SetRow` (per-set inputs: weight | reps | RPE chips | set-type chip | completion checkmark), `InlineRPEChipRow` (5 chips with long-press → `DecimalRPEPickerSheet`), `SetTypeChip` (cycling chip with system-color background per type), and `PreviousColumn` (the inline matching-intent prior-set display using `PreviousMatchingIntent` from plan `01-01`). Wires the rest-timer integration: tapping the set-row completion checkmark commits the set AND calls `restTimerEngine.start(seconds: prescribedRest, exerciseName:)`; tapping the next set's weight field calls `restTimerEngine.stop()`. Closes the user-visible session-logging half of SESS-01, SESS-02, SESS-03, SESS-04 (overlay integration), SESS-09 (signed weight), and the data path for ROUTINE-08 (intent-split history via the matching-intent query).

## Goal

Tap "Start Workout" on a routine row → `SessionLoggerView` is the new tab's content. Header shows elapsed time + exercise progress + workout-notes button (the notes UI lands in plan `04-03`). Body is a `List` of `SessionExerciseCard`s — each card showing the exercise name + column header row + a `SetRow` per `SetEntry`. Per-set row: weight TextField | reps TextField | `InlineRPEChipRow` | `SetTypeChip` | completion checkmark. The `PreviousColumn` renders inline between prescribed and actual columns, showing `"175 × 8 @ 8 (Mon)"` from the most recent matching-intent prior set or `"—"` when none.

Set-row commit (tap the trailing checkmark when `weight > 0` AND `reps > 0`) flips `SetEntry.isComplete = true`, writes `completedAt = Date.now`, and calls `restTimerEngine.start(seconds: sessionExercise.prescribedRestSeconds, exerciseName: sessionExercise.exercise?.name ?? "")`. The `RestTimerOverlay` (plan `02-03`) is mounted at the top of the session view; the overlay reads from the shared `engine`.

Auto-stop on next-set entry: when the user taps an empty weight or reps cell on a still-incomplete set, `restTimerEngine.stop()` fires before the keyboard surfaces.

Bodyweight signed weight (SESS-09): for exercises with `equipment == .bodyweight`, the weight TextField label changes to "Added (signed)" and accepts negative values (assisted machine, e.g. assisted pull-up).

## Requirements Covered

- **SESS-01** (`SessionFactory.start snapshots all routine prescription fields`) — the logger is the consumer; renders against the snapshotted SessionExercise fields, not the source Routine.
- **SESS-02** (`per-set logging: weight / reps / RPE / set-type / notes`) — the SetRow + InlineRPEChipRow + SetTypeChip + per-set notes button (notes sheet lives in plan `04-03`).
- **SESS-03** (`set inputs auto-populate from previous matching-intent session; inline "previous" column`) — `PreviousColumn` view + auto-populate behavior on empty-cell tap.
- **SESS-04** (rest timer integration — auto-start on set commit; auto-stop on next-set entry) — the SetRow's completion handler + the next-set-tap handler.
- **SESS-09** (bodyweight signed weight) — the SetRow detects `equipment == .bodyweight` and offers a signed numeric field.
- **SESS-11** (workout-level + pinned per-exercise notes inline) — the header notes button is wired (notes editing UI is plan `04-03`).
- **ROUTINE-08** (same-routine-different-intent produces separate per-exercise histories) — the data path is the matching-intent `PreviousColumn` query.

import SwiftUI
import SwiftData

public struct SessionLoggerView: View {
    @Environment(\.modelContext) private var ctx
    @Environment(\.dismiss) private var dismiss

    @Bindable public var session: Session
    @State private var engine = RestTimerEngine.makeProduction()
    @State private var elapsedStart: Date
    @State private var presentingFinishConfirm = false
    @State private var presentingDiscardConfirm = false

    public init(session: Session) {
        self.session = session
        self._elapsedStart = State(initialValue: session.startedAt)
    }

    public var body: some View {
        VStack(spacing: 0) {
            RestTimerOverlay(engine: engine)
                .padding(.top, 8)
            headerChips
            List {
                ForEach(sortedExercises) { se in
                    SessionExerciseCard(
                        sessionExercise: se,
                        engine: engine,
                        onCommitSet: { commitSet($0, for: se) },
                        onTapEmptyCell: { engine.stop() }
                    )
                }
            }
            .listStyle(.insetGrouped)
        }
        .navigationTitle("Workout")                                            // UI-SPEC verbatim
        .navigationBarTitleDisplayMode(.inline)
        .toolbar {
            ToolbarItem(placement: .principal) {
                VStack {
                    Text("Workout").font(.headline)
                    Text(session.routineSnapshotName)                          // UI-SPEC verbatim subtitle
                        .font(.caption).foregroundStyle(.secondary)
                }
            }
            ToolbarItem(placement: .topBarLeading) {
                if loggedSetCount == 0 {
                    Button("Discard") {                                        // UI-SPEC verbatim
                        presentingDiscardConfirm = true
                    }
                }
            }
            ToolbarItem(placement: .topBarTrailing) {
                Button("Finish") {                                              // UI-SPEC verbatim
                    presentingFinishConfirm = true
                }
                .foregroundStyle(Color.accentColor)
            }
        }
        .confirmationDialog(
            "Finish Workout?",                                                  // UI-SPEC verbatim
            isPresented: $presentingFinishConfirm,
            titleVisibility: .visible
        ) {
            Button("Finish") { finish() }                                       // UI-SPEC verbatim
            Button("Keep Logging", role: .cancel) {                             // UI-SPEC verbatim
                presentingFinishConfirm = false
            }
        } message: {
            Text("\(loggedSetCount) sets logged · \(elapsedLabel)")             // UI-SPEC verbatim format
        }
        .alert(
            "Discard Workout?",                                                 // UI-SPEC verbatim
            isPresented: $presentingDiscardConfirm
        ) {
            Button("Discard", role: .destructive) { discard() }                 // UI-SPEC verbatim
            Button("Cancel", role: .cancel) {                                   // UI-SPEC verbatim
                presentingDiscardConfirm = false
            }
        } message: {
            Text("No data will be saved.")                                      // UI-SPEC verbatim
        }
    }

    private var sortedExercises: [SessionExercise] {
        (session.exercises ?? []).sorted { $0.orderIndex < $1.orderIndex }
    }

    private var loggedSetCount: Int {
        sortedExercises.reduce(0) { sum, se in
            sum + (se.sets ?? []).filter { $0.isComplete }.count
        }
    }

    @ViewBuilder private var headerChips: some View {
        TimelineView(.periodic(from: elapsedStart, by: 1)) { _ in
            HStack(spacing: 12) {
                HStack(spacing: 4) {
                    Image(systemName: "clock")
                    Text(elapsedLabel)
                }
                Text("\(progressLabel)")                                        // "3 of 6"
                Button {
                    // Plan 04-03 — present WorkoutNotesSheet
                } label: {
                    HStack(spacing: 4) {
                        Image(systemName: "square.and.pencil")
                        Text("Notes")                                           // UI-SPEC caption label
                    }
                }
            }
            .font(.caption)
            .padding(.horizontal, 16)
            .padding(.vertical, 8)
        }
    }

    private var elapsedLabel: String {
        let s = max(0, Int(Date.now.timeIntervalSince(elapsedStart)))
        return String(format: "%d:%02d", s / 60, s % 60)
    }

    private var progressLabel: String {
        let totalExercises = sortedExercises.count
        let completed = sortedExercises.prefix(while: { se in
            (se.sets ?? []).allSatisfy { $0.isComplete }
        }).count
        return "\(min(completed + 1, totalExercises)) of \(totalExercises)"
    }

    private func commitSet(_ entry: SetEntry, for se: SessionExercise) {
        entry.isComplete = true
        entry.completedAt = .now
        try? ctx.save()
        let prescribed = max(1, se.prescribedRestSeconds)
        engine.start(seconds: prescribed, exerciseName: se.exercise?.name ?? "")
    }

    private func finish() {
        session.completedAt = .now
        session.totalDurationSeconds = Int(Date.now.timeIntervalSince(session.startedAt))
        engine.stop()
        try? ctx.save()
        dismiss()
    }

    private func discard() {
        engine.stop()
        ctx.delete(session)
        try? ctx.save()
        dismiss()
    }
}

import SwiftUI

public struct SetRow: View {
    @Bindable public var entry: SetEntry
    public let sessionExercise: SessionExercise
    public let onCommit: () -> Void
    public let onTapEmptyCell: () -> Void

    @State private var weightText: String = ""
    @State private var repsText: String = ""

    public init(
        entry: SetEntry,
        sessionExercise: SessionExercise,
        onCommit: @escaping () -> Void,
        onTapEmptyCell: @escaping () -> Void
    ) {
        self.entry = entry
        self.sessionExercise = sessionExercise
        self.onCommit = onCommit
        self.onTapEmptyCell = onTapEmptyCell
    }

    public var body: some View {
        HStack(spacing: 8) {
            Text(setLabel)                                                    // "1", "2", "W1" for warmup
                .font(.body)
                .frame(width: 32, alignment: .leading)
            PreviousColumn(
                exerciseID: sessionExercise.exercise?.id,
                intentRaw: sessionExercise.intentRaw
            )
            .frame(width: 80, alignment: .leading)
            TextField("—", text: $weightText)                                  // UI-SPEC verbatim placeholder
                .keyboardType(weightKeyboardType)
                .frame(width: 60)
                .onTapGesture {
                    if !entry.isComplete { onTapEmptyCell() }
                }
                .onChange(of: weightText) { _, newValue in
                    if let d = Double(newValue) { entry.weight = d }
                }
            TextField("—", text: $repsText)                                    // UI-SPEC verbatim placeholder
                .keyboardType(.numberPad)
                .frame(width: 40)
                .onTapGesture {
                    if !entry.isComplete { onTapEmptyCell() }
                }
                .onChange(of: repsText) { _, newValue in
                    if let i = Int(newValue) { entry.reps = i }
                }
            InlineRPEChipRow(rpe: Binding(
                get: { entry.rpe },
                set: { entry.rpe = $0 }
            ))
            SetTypeChip(setTypeRaw: Binding(
                get: { entry.setTypeRaw },
                set: { entry.setTypeRaw = $0 }
            ))
            Spacer()
            Button {
                if entry.weight > 0 && entry.reps > 0 {
                    onCommit()
                }
            } label: {
                Image(systemName: entry.isComplete ? "checkmark.circle.fill" : "circle")
                    .foregroundStyle(entry.isComplete ? Color.accentColor : Color.secondary)
                    .font(.title2)
                    .frame(minWidth: 44, minHeight: 44)                        // UI-SPEC HIG
            }
            .buttonStyle(.plain)
            .accessibilityLabel(
                entry.isComplete
                ? "Set \(entry.orderIndex + 1) complete"
                : "Mark set \(entry.orderIndex + 1) complete"
            )
        }
        .padding(.vertical, 12)                                                // UI-SPEC md
        .onAppear {
            weightText = entry.weight == 0 ? "" : String(entry.weight)
            repsText = entry.reps == 0 ? "" : String(entry.reps)
        }
    }

    private var setLabel: String {
        let n = entry.orderIndex + 1
        return entry.isWarmup ? "W\(n)" : "\(n)"
    }

    private var weightKeyboardType: UIKeyboardType {
        // SESS-09 — bodyweight equipment supports signed numeric.
        if sessionExercise.exercise?.equipment == .bodyweight {
            return .numbersAndPunctuation
        }
        return .decimalPad
    }
}

import SwiftUI

public struct SetTypeChip: View {
    @Binding public var setTypeRaw: String

    public init(setTypeRaw: Binding<String>) {
        self._setTypeRaw = setTypeRaw
    }

    public var body: some View {
        Button {
            cycleType()
        } label: {
            if let type = SetType(rawValue: setTypeRaw), type != .working {
                Text(label(for: type))
                    .font(.caption)
                    .fontWeight(.semibold)
                    .padding(.horizontal, 8)
                    .padding(.vertical, 4)
                    .background(systemColor(for: type))
                    .foregroundStyle(.white)
                    .clipShape(Capsule())
            } else {
                EmptyView()                                                    // working sets render no chip
            }
        }
        .buttonStyle(.plain)
        .accessibilityLabel(label(for: SetType(rawValue: setTypeRaw) ?? .working))
        .accessibilityHint("Tap to cycle set type. Tap and hold for menu")     // UI-SPEC verbatim a11y
        .contextMenu {
            Button("Working") { setTypeRaw = SetType.working.rawValue }        // UI-SPEC verbatim
            Button("Warm-up") { setTypeRaw = SetType.warmup.rawValue }
            Button("Drop Set") { setTypeRaw = SetType.drop.rawValue }
            Button("To Failure") { setTypeRaw = SetType.failure.rawValue }
            Button("Rest-Pause") { setTypeRaw = SetType.restPause.rawValue }
        }
    }

    private func cycleType() {
        let order: [SetType] = [.working, .warmup, .drop, .failure, .restPause]
        let current = SetType(rawValue: setTypeRaw) ?? .working
        let idx = order.firstIndex(of: current) ?? 0
        setTypeRaw = order[(idx + 1) % order.count].rawValue
    }

    private func label(for type: SetType) -> String {
        switch type {
        case .warmup: return "warmup"
        case .working: return "Working"
        case .drop: return "drop"
        case .failure: return "failure"
        case .restPause: return "rest-pause"
        }
    }

    private func systemColor(for type: SetType) -> Color {
        // UI-SPEC § Set-type chip system colors
        switch type {
        case .warmup: return Color(.systemBlue)
        case .drop: return Color(.systemOrange)
        case .failure: return Color(.systemRed)
        case .restPause: return Color(.systemPurple)
        case .working: return .clear
        }
    }
}

import SwiftUI
import SwiftData

public struct PreviousColumn: View {
    @Environment(\.modelContext) private var ctx
    public let exerciseID: UUID?
    public let intentRaw: String
    @State private var hint: PreviousMatchingIntentHit?

    public init(exerciseID: UUID?, intentRaw: String) {
        self.exerciseID = exerciseID
        self.intentRaw = intentRaw
    }

    public var body: some View {
        Group {
            if let hit = hint {
                Text(formatHint(hit))
                    .font(.caption)
                    .foregroundStyle(.tertiary)
            } else {
                Text("—")                                                       // UI-SPEC verbatim placeholder
                    .font(.caption)
                    .foregroundStyle(.tertiary)
            }
        }
        .task {
            hint = PreviousMatchingIntent.fetchTopWorkingSet(
                exerciseID: exerciseID,
                intentRaw: intentRaw,
                context: ctx
            )
        }
    }

    private func formatHint(_ hit: PreviousMatchingIntentHit) -> String {
        let day = hit.sessionStartedAt.formatted(.dateTime.weekday(.abbreviated))
        let rpeText = hit.rpe.map { String(format: "@ %.1f", $0) } ?? ""
        let w = hit.weight.truncatingRemainder(dividingBy: 1) == 0
            ? "\(Int(hit.weight))"
            : String(format: "%.1f", hit.weight)
        // UI-SPEC verbatim: "175 × 8 @ 8 (Mon)"
        return "\(w) × \(hit.reps) \(rpeText) (\(day))".replacingOccurrences(of: "  ", with: " ")
    }
}

## Anti-Patterns to Avoid

- Do NOT compute "previous" inside SetRow body. RESEARCH § Anti-Patterns to Avoid: the query fires once per row in `.task`, cached in `@State`. Body re-renders read from cache.
- Do NOT mutate `RoutineExercise` rows from the logger. The logger reads/writes `SessionExercise`/`SetEntry` only. PITFALLS-doc #1.
- Do NOT use `Timer.publish` for elapsed time. `TimelineView(.periodic(from: elapsedStart, by: 1))` re-renders the elapsed chip without drift.
- Do NOT call `engine.start(...)` synchronously on UI events without `try? ctx.save()` first. The committed set must persist before the rest period kicks off.
- Do NOT pass the entire `Session` into `SessionExerciseCard`. Pass the `SessionExercise` row only. Defensive scoping.
- Do NOT skip the `if entry.weight > 0 && entry.reps > 0` guard on commit. Without it, an empty checkmark-tap would flip `isComplete = true` with zero weight/reps — corrupting the matching-intent query in future sessions.
- Do NOT render the `SetTypeChip` capsule for working sets. UI-SPEC: "working sets render no chip at all." The chip's `body` returns `EmptyView()` for `.working`.
- Do NOT compute `weight: Double` via `String(entry.weight)` for the TextField initial value — that renders "0.0" for empty sets. Use `entry.weight == 0 ? "" : String(entry.weight)` so the placeholder renders correctly.
- Do NOT call `engine.stop()` on every keystroke — only on the empty-cell tap (the moment the user starts editing the next set). Hammering stop would re-cancel the lock-screen notification repeatedly.
- Do NOT navigate to `SessionLoggerView` via a `.sheet`. The session-logger is full-screen; use `NavigationStack` push so the back button + the system gesture work natively.
